Flushing the Server Cache

You can flush the server cache for the following types of entries.

  • Zimlets

  • Themes

  • Locale

When you add or change themes (skins) properties files and local resource files or update a customized Zimlet, you must flush the cache to reload the new content. Until you do this, the new themes, locales, or Zimlets are not available on the server.

Procedure

  1. Go to Configure>Server

  2. Select the server and in the gear icon click Flush Cache.

Flushing the cache might cause a temporary slowdown of mail delivery.
