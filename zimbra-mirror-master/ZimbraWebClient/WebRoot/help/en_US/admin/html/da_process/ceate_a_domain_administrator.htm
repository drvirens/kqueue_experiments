Create a Domain Administrator

  1. Go to Manage>Accounts.

  2. In the gear icon, select New.

  3. Create the account.

  4. If the domain administrator's role is basic domain administration, check Administrator and check Assign Default domain administrator views and rights.

  5. You can give domain administrators the ability to set the quota for individual accounts. In the Account quota field enter the maximum quota the domain administrator can set for any account.  

  6. Complete the account create and click Finish.

 

  • To create a quick access link to the administration console from the domain administrator's Zimbra Web Client account, run the following zmprov CLI

 

zmprov md < server.domainexample.com> zimbraWebClientAdminReference <https:// server.domainexample.com:7071/
