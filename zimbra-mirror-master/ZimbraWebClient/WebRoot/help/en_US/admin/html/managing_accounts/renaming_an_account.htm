Renaming an Account

You can rename an account, changing the account name and other general information.  The renamed account keeps the same ID as the original account.  All messages, contacts, folders, calendar, etc. remain the same.

  1. Go to Manage>Accounts.

  2. In the Content pane, select the account to rename and in the gear icon select Edit.

  3. Change the account name and other information if necessary.

  4. Click Save.  The new account name is displayed in the Account list.

 

After accounts are renamed, you should immediately run a full backup of the renamed account.
